Mini JavaScript Exercises

Each tile is a small refresher, challenge, and/or exercise focusing on vanilla JS. Projects are small and intentionally limited to one HTML page. Project ideas are from Wes Bos. This project is currently ongoing and optimized for Chrome.

This site is from HTML5 UP... I've been looking for a reason to work with one of the templates.

Sound Garden

Client: Mini wind chime sound effects controlled by the keyboard.

Dev: JS and ES6 set event listeners, trigger sound effects, and add/remove a CSS class.

A Clock

Client: A pink analog clock and a digital clock.

Dev: JS and ES6 capture the current time, convert time to degrees, and rotate the clock hands based on a timer.

CSS Variables

Client: Live stylistic changes to a (famous) painting.

Dev: JS listeners connect to HTML inputs that change CSS variables.

Art Historian: Serusier and Synthetism are the bomb.

Flex Panels

Client: Interactive way to display visual content and minimal text.

Dev: Use JS to toggle flexbox classes.

Art Historian: The Pre-Raphaelites: "To produce thoroughly good pictures and statues"

Filter API

Client: Search box for Philly public art works

Dev: Use the fetch method to get API data and then compare data to user input and render matches on the document.

HTML5 Canvas

Client: Relaxing canvas reminiscent of a pared down Microsoft paint.

Dev: Using the HTML5 canvas, capture coordinates of user clicks to display drawing.

Art Historian: "You're a regular surrealist automatist"
